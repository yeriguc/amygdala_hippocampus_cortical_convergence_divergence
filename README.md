# amygdala_hippocampus_cortical_convergence_divergence
This repository will contain analysis code for:

Erigüç et al., Selective convergence and graded divergence of hippocampal and amygdalar subregions using functional connectivity.

The code is currently being cleaned and documented for public release. It will be added here after preprint publication.

Raw neuroimaging data were obtained from the Human Connectome Project Young Adult dataset and are not redistributed here.
